# Pre-Task Checklist

> Run before starting any implementation task.

## Context Loading

- [ ] Read `agent.md` for relevant docs to load
- [ ] Load docs/rules/ applicable to this task type
- [ ] Load docs/checklists/code-review.md (know the review criteria upfront)

## Task Understanding

- [ ] What is the specific deliverable? (not the goal, the concrete output)
- [ ] Which files will be created or modified?
- [ ] What existing code does this interact with?
- [ ] What are the acceptance criteria?

## Constraints

- [ ] Relevant coding rules from docs/rules/ reviewed
- [ ] Known failure patterns from docs/memory/failure-patterns.md checked
- [ ] Test approach determined (TDD / integration / etc.)

## Planning

- [ ] Implementation approach decided
- [ ] Edge cases identified
- [ ] No assumptions left unchecked
