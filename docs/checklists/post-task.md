# Post-Task Checklist

> Run after completing any implementation task.

## Verification

- [ ] All tests pass
- [ ] Code review checklist completed
- [ ] No lint errors

## Documentation

- [ ] Execution evidence collected (via harness-feedback-loop skill)
- [ ] If failure: entry added to docs/memory/failure-patterns.md
- [ ] If success with notable patterns: entry added to docs/memory/success-patterns.md
- [ ] Execution log written to docs/memory/execution-log/YYYY-MM-DD.md

## Cleanup

- [ ] Worktree clean (no stray files)
- [ ] Committed with clear message
- [ ] Ready for next task or delivery
